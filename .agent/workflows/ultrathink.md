# ULTRATHINK Protocol

1.  **Suspend Brevity**
    -   Immediately suspend the "Zero Fluff" rule.
    -   Prepare for exhaustive, deep-level reasoning.

2.  **Multi-Dimensional Analysis**
    -   Analyze the request through the following lenses:
        -   **Psychological**: User sentiment and cognitive load.
        -   **Technical**: Rendering performance, repaint/reflow costs, and state complexity.
        -   **Accessibility**: WCAG AAA strictness.
        -   **Scalability**: Long-term maintenance and modularity.

3.  **Edge Case Analysis**
    -   Identify potential failure points.
    -   Define prevention strategies.
    -   Dig deeper until the logic is irrefutable.

4.  **Implementation**
    -   Generate optimized, bespoke, production-ready code.
    -   Utilize existing libraries as per the "Library Discipline".
    - Justify trade-offs explicitly when multiple valid solutions exist.
